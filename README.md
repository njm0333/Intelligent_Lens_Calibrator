
# Intelligent\_Lens\_Calibrator

OpenCV 기반의 분석을 통해 카메라의 고유 파라미터를 산출하고, 광각 렌즈의 왜곡을 보정하는 영상 처리 시스템입니다.

## 1\. 주요 기능 

컴퓨터 비전 기술을 활용하여 2D 이미지 좌표와 3D 세계 좌표 사이의 수학적 관계를 규명합니다.

  - **Intrinsic Parameter Extraction**: 체스보드 패턴 인식을 통해 초점 거리($f_x, f_y$)와 주점($c_x, c_y$)이 포함된 카메라 매트릭스($K$)를 산출합니다.
  - **Distortion Modeling**: 렌즈 곡률로 인해 발생하는 방사상(Radial) 및 접선(Tangential) 왜곡 계수를 산출합니다.
  - **Real-time Rectification**: 추출된 파라미터를 기반으로 연산을 수행하여 기하학적 왜곡이 제거된 영상을 실시간으로 렌더링합니다.

## 2\. 알고리즘 구현 및 예시영상 분석 결과

|**Focal Length**|초점 거리 ($f_x, f_y$)|**956.50 / 962.04**|

## 3\. 알고리즘의 한계 및 고찰

  - **데이터 품질 중심의 샘플링**: 이미지의 수량(Quantity)보다 정확한 코너 검출이 가능한 선명한 이미지(Quality)가 RMS Error 감소에 결정적임을 확인하였습니다. (울툴불퉁한 면에서 찍은 영상 16장보다, 평평한 면에서 찍은 영상 13장으로 선별 후 오차 대폭 감소)
  - **포즈 다양성(Pose Diversity)**: 렌즈 왜곡 계수의 정확도를 높이기 위해 체스보드를 화면의 중앙뿐만 아니라 상하좌우 외곽 경계선)에 배치하여 샘플을 채취하였습니다.
  - **평면성 유지**: 보정의 신뢰도를 높이기 위해 패턴지를 평평한 판에 부착하여 종이의 굴곡으로 인한 기하학적 왜곡 간섭을 최소화하였습니다.
